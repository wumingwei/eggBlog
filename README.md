
### start a Blog system using  egg.js  from zero.


#### Installation

```bash

$ npm install egg --save


Node.js >= 8.0.0 required.
```

#### Getting Started

Follow the commands listed below.

```bash

$ git clone https://github.com/wumingwei/eggBlog.git
$ npm install egg-init -g

$ cd eggBlog
$ npm install
$ npm run dev

```
